# growi-feed-converter

Growi の API から RSS フィードを出力するアプリ。

## 使い方

```text
http://[このスクリプトを設置しているホスト名]?url=https://demo.growi.org&limit=10&title=Growi&description=Growi%20RecentChanges
```

| パラメータ  | 意味         |
| ----------- | ------------ |
| url         | Growi の URL |
| limit       | 表示件数     |
| title       | RSS 名       |
| description | 説明文       |
